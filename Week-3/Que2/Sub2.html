Box Dimensions Calculation:

  • Content width: 300px
  • Padding: 20px on each side (20px × 2 = 40px)
  • Border: 5px on each side (5px × 2 = 10px)
  • Margin: 15px on each side (15px × 2 = 30px)
  • Total Box Width: 300px (content) + 40px (padding) + 10px (border) + 30px (margin) = 380px
